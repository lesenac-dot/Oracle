# Oracle migration consolidated

Este repositório agrupa temporariamente o conteúdo disperso em vários repositórios do usuário para facilitar acesso via celular.

Estrutura criada:
- oracle-core/
- oracle-tuning/
- oracle-labs/
- oracle-scripts/
- oracle-docs/
- migration-scripts/ (scripts para quando for migrar para repositórios independentes no PC)

Cada diretório contém arquivos copiados de seus repositórios origem. Este é um snapshot sem histórico — caso deseje preservar histórico, execute os scripts em migration-scripts/ num PC.
